# UI Tokens

Tokens de inyección compartidos exportados por `@libs/tokens`.

## Exportaciones

- `DISABLE_ON_LOADING`: `InjectionToken<Signal<boolean>>`

## Propósito

Permite deshabilitar controles interactivos (wrappers/input wrappers) durante estados de carga de forma transversal.

## Ubicación

- `libs/ui/src/lib/tokens/disable-on-loading.token.ts`

## Regla

Todo token nuevo debe incluir tipo explícito y caso de uso; documentarlo aquí cuando sea parte del contrato público.

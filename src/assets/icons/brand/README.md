# Brand Icon Set (Bookmarks + Rocket)

Sistema visual comun:
- `viewBox="0 0 24 24"`
- `stroke="currentColor"`
- `stroke-width="1.6"`
- `stroke-linecap="round"`
- `stroke-linejoin="round"`
- Sin rellenos complejos para mejor nitidez en tamanos pequenos.

2. `bookmarks-orbit-node.svg` - Nave orbitando un link
3. `bookmarks-rocket-shield.svg` - Nave dentro de escudo
4. `bookmarks-star-bookmark.svg` - Bookmark + estrella
5. `bookmarks-link-constellation.svg` - Red de links + nave
6. `bookmarks-b-orbit.svg` - Letra B orbital
7. `bookmarks-folder-trail.svg` - Carpeta + estela
8. `bookmarks-portal-bookmark.svg` - Portal + marcador
